WhatsUpGold 2018

Ipswitch의 WhatsUp® Gold 2018은 한층 더 Upgrade된 맵 토폴로지와 다이내믹 리포트를 제공합니다.
IT Infra에 대한 정보를 수집하고 분석하여, 전체적인 IT Infra의 상태를 한눈에 확인할 수 있으며, 장애의 신속한 인지 및 해결을 도와드립니다.

  • 1.통합적인 MAP과 필터링 기능

    ARP를 통해 DEVICES 간의 연결관계를 자동으로 감지하여MAP을 보여줍니다. 링크 연결 관계 외에도, 인터페이스 상태 / AP장치와 CLIENT 장비 간 연결 상태 / 가상 머신의 HOST와 GUEST간 연결 상태에 대해서 한 눈에 파악할 수 있습니다. 또한, 필터링 기능을 제공하여, DEVICES의 IP/OS/상태/정보수집 방식 등 특징에 따라, 특정 DEVICES를 확인할 수 있습니다.

  • 2.다양한 방식으로 데이터 수집

    SNMP V1/2/3을 비롯하여, WMI / ADO / VMWARE / JMX / SMIS / AWS 등 10여가지의 다양한 방식을 통해 다각적인 모니터링을 데공합니다. 따라서, 물리적인 서버뿐만 아니라, 가상 머신, 스토리지, 어플리케이션, 데이터베이스 등에 대해 통합적으로 모니터링 할 수 있습니다.

  • 3.풍부한 모니터링 라이브러리

    DEVICES의 UP / DOWN상태 / CPU / DISK / 메모리/ INTERFACE TRAFFIC 을 포함하여, 50여개의 모니터링 라이브러리를 제공 합니다. PROCESS MONITOR를 통해 주요 프로세스의 UP / DOWN을 체크하여, 프로그램 작동 에러를 빠르게 감지할 수 있으며, HTTP CONTENTS MONITOR를 통해 웹 페이지의 UP / DOWN을 모니터링 하여, 웹 페이지 DOWN 시, 신속하게 대처할 수 있습니다.

  • 4. 그래픽적인 리포트 템플릿

    Up /Down상태 / CPU / Disk / Memory / Interface Traffic 등 다양한 요소에 대한 100여 개의 리포트 템플릿을 제공하며, 원하는 항목만으로 편집이 가능합니다. 특히, WhatsUp Gold의 리포트는 그래픽적인 요소가 탁월하기 때문에, 다양한 그래프와 차트를 통해데이터의 변화 추이 및 비율 변화를 한 눈에 파악할 수 있습니다.

  • 5.사용자 중심의 대시보드 구성

    관리 목적에 따라 리포트를 선택하고, 원하는 곳에 위치시킴으로써,사용자 입맛에 맞추어 유연하게 대시보드를 구성할 수 있습니다. 또한, Export 기능을 통해 PDF/Excel 등의 형태로 ‘내보내기’(전환)가 가능하여,Infra 관련 보고서를 보다 간편하게 작성할 수 있도록 도와줍니다.

  • 6.Up/Down 및 성능에 따른 알람

    Up / Down / Maintenance / Unknown과 같은 Devices의 상태 변화 및 경과 시간에 따라, 팝업 메시지 / SMS / E-mail/ 로그 등 19가지 형태로 알람을 받을 수 있으며, 여러 형태의 알람을 조합하여 알람 정책을 구성할 수도 있습니다. 또한, 성능 수치에 임계치를 설정하여, 모니터링한 수치가 임계치에서 벗어난 경우, E-mail과 SMS로 알람을 받을 수 있습니다.

WhatsUpGold Process

WhatUp Gold는 수 년간 연구한 poller(데이터 수집 장치)를 기반으로 SNMP, WMI, SSH 등 다양한 프로토콜을 통해 정보를 수집합니다.
이렇게 수집된 데이터는 Microsoft SQL Server(DB)에 저장되며, 분석 및 가공되어 이해하기 쉬운 형태의 리포트로 제공됩니다.
사용자는 IIS 기반 웹 브라우저에서 리포트들을 이용하여 입맛에 맞게 화면(대시보드)을 구성할 수 있으며, 이를 통해 IT Infra에 대한 통합적이고, 효과적인 관제가 가능합니다.
